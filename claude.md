Use Tailwind CSS para estilizar este componente. Evite estilos inline. Use espaçamento consistente baseado em rem.
Tipografia: Não use Inter ou Roboto. Escolha uma fonte sans-serif moderna como Montserrat ou Poppins para o corpo e uma fonte serifada marcante para títulos.
Cores: Use uma paleta coesa de tons terrosos com apenas uma cor de destaque vibrante em botões CTA.
Backgrounds: Adicione um gradiente sutil ou um padrão de ruído (noise texture) leve no fundo para dar profundidade.
